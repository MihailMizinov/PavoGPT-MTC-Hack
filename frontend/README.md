# Chainlit Lua-генератор 

Чат-приложение на Chainlit, которое принимает текстовые запросы, обрабатывает прикреплённые файлы (PDF, DOCX, TXT) и генерирует **Lua-код** в качестве ответа. 

## Возможности

- Чат с историей в рамках сессии.
- Загрузка файлов: `.pdf`, `.docx`, `.txt` – их содержимое извлекается и добавляется в контекст.
- Настройки в боковой панели: выбор модели (заглушка) и включение/отключение имитации «Мышления».
- Стриминг Lua-кода (побуквенное появление).
- Визуальные шаги (`cl.Step`) для чтения документов и «Мышления».
- Кастомный CSS (фон, стили сообщений, кнопка отправки).

## Установка и запуск

### 1. Клонируйте репозиторий и перейдите в папку проекта

```bash
git clone <your-repo-url>
cd MtcHackCleanKoders
```

### 2. Создайте виртуальное окружение (рекомендуется)

```bash
python -m venv .venv
.venv\Scripts\activate      # Windows
# source .venv/bin/activate # Linux/macOS
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Убедитесь, что папка `public` с файлом `custom.css` находится в корне проекта

Пример структуры:
```
MtcHackCleanKoders/
├── app.py
├── requirements.txt
├── README.md
└── public/
    └── custom.css
```

### 5. Запустите приложение

```bash
chainlit run app.py -w
```

После запуска откройте браузер по адресу `http://localhost:8000`.

## Настройка внешнего вида

Вы можете изменить стили в `public/custom.css`. Пример уже содержит:

- Градиентный фон чата.
- Скруглённые сообщения с разными цветами для пользователя и ассистента.
- Тёмно-бардово-розовую кнопку отправки.
